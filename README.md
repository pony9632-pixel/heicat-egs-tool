# STUDIO A 黑貓宅急便工具

統一速達（黑貓宅急便）EGS 企業帳號的桌面建單工具，僅支援 macOS。

---

## 功能總覽

| 功能 | 說明 |
|---|---|
| **單筆建單** | 填表即可產生 PDF 託運單，姓名欄輸入自動比對通訊錄 |
| **批次建單** | 上傳 CSV 一次建立多筆，顯示進度、失敗不中斷 |
| **門市通訊錄** | 管理門市收件人，支援搜尋、複選刪除、CSV 匯入匯出 |
| **廠商通訊錄** | 與門市分頁獨立管理廠商聯絡人 |
| **自動更新** | 每次啟動自動比對 GitHub 版本，有新版靜默更新 |

---

## 給同事的快速安裝

### 步驟 1：安裝 Python（沒裝過才需要）

前往 **https://www.python.org/downloads/** 下載最新版 Python 3，執行安裝檔，全部選預設按「繼續」即可。

> 已經有 Python 3.10 以上的電腦可以跳過這步。

### 步驟 2：一鍵安裝工具

開「**終端機**」（Spotlight 搜尋 `Terminal`），貼上以下指令後按 Enter，等它跑完即可：

```bash
curl -fsSL "https://raw.githubusercontent.com/pony9632-pixel/heicat-egs-tool/main/install.sh" | bash
```

腳本會自動下載程式、建立執行環境、安裝套件，完成後自動開啟程式資料夾。

### 步驟 2：填設定

雙擊 `啟動黑貓工具.command`，切到「**設定**」分頁，填：

- **客戶代號**（黑貓業務給的）
- **API 授權碼**（黑貓業務給的）
- **寄件人資料**（門市名稱、電話、地址）

按「**儲存設定**」→ 按「**測試 API 連線**」確認顯示 ● 連線正常。

完成！可以開始建單了。

> ⚠️ 第一次雙擊 `啟動黑貓工具.command` 若出現「無法打開，因為來自未識別的開發者」：  
> **系統設定 → 隱私權與安全性** → 找到被封鎖項目 → 點「**仍要打開**」。

## 日常使用

### 單筆建單

1. 「**建立寄件單**」分頁
2. 在姓名欄輸入文字 → 自動比對通訊錄，點選即填入電話與地址
3. 或按「**從通訊錄**」挑選，或「**從剪貼板帶入**」
4. 選包裹明細（尺寸、溫層、日期），可點「今天 / 明天 / 後天」快速填入
5. 按「**建立寄件單 →**」
6. PDF 自動加入待列印清單，同時存入 `黑貓單號/` 資料夾

### 批次建單

1. 「**批次建單**」分頁
2. 第一次點「**產生 CSV 範本**」→ 用 Excel 填寫資料
3. 存檔後，回程式按「**載入 CSV**」選檔案
4. 確認統計卡（有效筆數）無誤 → 按「**全部建單 →**」
5. 畫面顯示進度「建單中 x / y 筆」，完成後自動開啟 `黑貓單號/` 資料夾
6. 每筆結果同時寫入 `黑貓單號/build_log.txt` 保留紀錄

### 通訊錄

- **門市通訊錄** / **廠商通訊錄** 分頁獨立管理
- 勾選多筆後，點「**刪除選取 (N)**」一次刪除
- 搜尋欄右側的 × 可快速清除搜尋
- 「**匯出 CSV**」→ 用 Excel 批次編輯 →「**匯入 CSV**」更新

## 鍵盤快捷鍵

| 按鍵 | 功能 |
|---|---|
| ⌘1 / ⌘2 / ⌘3 / ⌘4 | 切換到 建立寄件單 / 批次建單 / 通訊錄 / 設定 |
| ⌘V / ⌘C / ⌘X / ⌘A | 貼上 / 複製 / 剪下 / 全選（注音模式也適用） |
| 右鍵 / Ctrl+點擊 | 在欄位上顯示「剪下／複製／貼上／全選」選單 |

## 自動更新

每次啟動時自動到 GitHub 檢查新版，有更新即下載並重啟，不需手動操作。  
`config.yaml`（設定）和 `contacts.json`（通訊錄）在更新時完整保留。

## 常見問題

<details>
<summary><b>一鍵安裝中斷怎麼辦</b></summary>

重新執行安裝指令即可，會自動覆蓋更新，設定與通訊錄不受影響。

</details>

<details>
<summary><b>雙擊 .command 顯示「沒有適當的取用權限」</b></summary>

開「終端機」貼上：

```bash
chmod +x ~/Desktop/黑貓宅急便工具/黑貓主程式/啟動黑貓工具.command
```

重新雙擊即可。

</details>

<details>
<summary><b>雙擊 .command 跳出後立刻關掉</b></summary>

在終端機執行：

```bash
python3 -c "import yaml, pypdf; print('OK')"
```

沒顯示 `OK` 就重新執行一鍵安裝指令補裝套件。

</details>

<details>
<summary><b>建單失敗顯示 E009</b></summary>

請到「**設定**」分頁按「**儲存設定**」，讓設定檔寫入固定的品名類別值，再重試。

</details>

<details>
<summary><b>客戶代號 / API 授權碼怎麼取得</b></summary>

聯絡黑貓宅急便業務窗口，申請 EGS 企業帳號的 API 資訊。

</details>

<details>
<summary><b>PDF 存到哪裡</b></summary>

`黑貓單號/` 資料夾，與 `黑貓主程式` 同一層。  
檔名格式：`訂單號_OBT號碼.pdf`

</details>

## 隱私與資料

- `config.yaml`（API 帳號）只存在本機，不會上傳
- `contacts.json`（通訊錄）只存在本機，不會上傳
- 自動更新只下載程式碼，不會異動資料檔

此工具僅供 STUDIO A 內部使用。
