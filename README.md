# 黑貓宅急便 企業建單工具

統一速達（黑貓）EGS 企業帳號的桌面建單工具，支援單筆建單、批次 CSV 建單、通訊錄管理，建單後直接輸出 PDF。

---

## 安裝步驟（第一次使用）

### 1. 下載程式

點此下載：[黑貓工具 v1.0.0 下載](https://github.com/pony9632-pixel/heicat-egs-tool/releases/latest)

進入頁面後，點 **Source code (zip)** 下載，解壓縮後放到桌面。

### 2. 安裝 Python 3

前往 [python.org/downloads](https://www.python.org/downloads/) 下載安裝，一路按 Continue / Install。

> 已安裝過可跳過此步驟。確認方式：開終端機輸入 `python3 --version`，有顯示版本號即可。

### 3. 安裝套件

開啟「終端機」（Spotlight 搜尋 Terminal），貼上以下指令執行：

```bash
python3 -m pip install pyyaml pypdf
```

### 4. 啟動程式

進入解壓縮後的資料夾，雙擊 `黑貓主程式` 裡的 `啟動黑貓工具.command`。

> 首次執行若出現「無法打開」的安全性警告，請到  
> **系統設定 → 隱私權與安全性** → 點「仍要開啟」。

### 5. 填入設定

程式開啟後，切換到「**設定**」頁籤，填入：
- 客戶代號
- API 授權碼
- 寄件人（門市）資料

填完後按「**儲存設定**」，即可開始建單。

---

## 功能說明

| 頁籤 | 說明 |
|------|------|
| 單筆建單 | 填寫一筆收件人資料，立即建立 PDF 託運單 |
| 批次建單 | 上傳 CSV 檔案，一次建立多筆訂單 |
| 通訊錄 | 管理常用收件人，建單時一鍵帶入 |
| 設定 | 修改 API 帳號資料與寄件人資料 |

PDF 檔案會自動存到 `黑貓單號/` 資料夾，檔名格式為 `訂單號_OBT號碼.pdf`。

## 更新程式

程式啟動時會自動在背景檢查是否有新版本。  
若有更新，頂部會出現橘色通知列，**點一下即自動更新並重新啟動**，不需要手動下載。

## 常見問題

**Q：雙擊 `啟動黑貓工具.command` 沒有反應或立刻關閉？**  
請確認步驟 2、3 都完成（Python 已安裝、套件已安裝）。安裝套件請務必用 `python3 -m pip install pyyaml pypdf`，不要用 `pip3`。

**Q：建單失敗，出現 E009？**  
請到「設定」頁重新選擇「品名類別」後儲存，再重試。

**Q：客戶代號和 API 授權碼在哪裡拿？**  
請聯繫黑貓業務取得 EGS 企業帳號資料。

**Q：PDF 存在哪裡？**  
`黑貓單號/` 資料夾（和 `黑貓主程式` 同層）。

## 注意事項

- 此工具僅供公司內部使用
