# STUDIO A 黑貓宅急便工具

統一速達（黑貓宅急便）EGS 企業帳號的桌面建單工具。

- **單筆建單**：填一張表 → 立刻產生 PDF 託運單
- **批次建單**：上傳 CSV → 一次建立多筆
- **通訊錄**：常用收件人一鍵帶入
- 全自動 PDF 輸出、自動版本更新、支援中文輸入法的複製貼上

只在 macOS 上測過（其他系統未支援）。

---

## 給同事的快速安裝

> 如果你的同事已經裝好，跳到 [日常使用](#日常使用)。

### 步驟 1：下載

點以下連結，按 **「Source code (zip)」** 下載：

**👉 [最新版下載](https://github.com/pony9632-pixel/heicat-egs-tool/releases/latest)**

下載完成後解壓縮，整個資料夾拖到「桌面」或任何你方便找到的地方。

### 步驟 2：安裝 Python 3（裝過可跳）

到 [python.org/downloads](https://www.python.org/downloads/) 下載安裝，一路按 Continue / Install。

確認方式：開「終端機」（Spotlight 搜尋 `Terminal`）輸入：

```bash
python3 --version
```

有顯示 `Python 3.x.x` 就 OK。

### 步驟 3：安裝套件

在終端機貼上：

```bash
python3 -m pip install pyyaml pypdf requests
```

按 Enter，等它跑完。

### 步驟 4：第一次啟動

進到剛剛解壓縮的資料夾，**雙擊** `黑貓主程式/啟動黑貓工具.command`。

> ⚠️ 第一次可能跳「無法打開，因為來自未識別的開發者」：  
> **系統設定 → 隱私權與安全性** → 拉到下面找到「`啟動黑貓工具.command` 已封鎖」→ 點「**仍要打開**」。

### 步驟 5：填設定

切到「**設定**」分頁，填：
- **客戶代號**（黑貓業務給的）
- **API 授權碼**（黑貓業務給的）
- **寄件人資料**（門市名稱、電話、地址）

按「**儲存設定**」→ 按「**測試 API 連線**」確認顯示 ● 連線正常。

完成！可以開始建單了。

## 日常使用

### 單筆建單

1. 打開程式 → 「**建立寄件單**」分頁
2. 填收件人資料（也可以按「從通訊錄」帶入）
3. 選包裹尺寸、溫層
4. 按「**建立寄件單**」
5. PDF 自動產生並打開，存在 `黑貓單號/` 資料夾

### 批次建單

1. 「**批次建單**」分頁
2. 第一次用點「**產生 CSV 範本**」→ Excel 打開填資料
3. 填完後存檔，回程式按「**載入 CSV**」選檔案
4. 確認列表沒有警示行 → 按「**開始批次建單**」
5. 完成後自動打開 `黑貓單號/` 資料夾

### 通訊錄

- 「**通訊錄**」分頁可以新增 / 編輯 / 刪除常用收件人
- 點任一聯絡人 → 右邊看詳細 → 「**用此地址建單 →**」直接帶入單筆建單

## 鍵盤快捷鍵

| 按鍵 | 功能 |
|---|---|
| ⌘1 / ⌘2 / ⌘3 / ⌘4 | 切換到 建立寄件單 / 批次 / 通訊錄 / 設定 |
| ⌘V / ⌘C / ⌘X / ⌘A | 貼上 / 複製 / 剪下 / 全選（注音模式也可用） |
| 右鍵 / Ctrl+點擊 | 在任何欄位上顯示「剪下／複製／貼上／全選」選單 |

## 自動更新

每次啟動程式時會自動到 GitHub 檢查有沒有新版。  
有新版會直接下載、覆蓋、重啟，**不會問你**。`config.yaml` 和 `contacts.json` 都會保留。

如果不想等啟動，也可以直接重新下載最新版覆蓋整個 `黑貓主程式` 資料夾。

## 常見問題

<details>
<summary><b>雙擊 .command 顯示「沒有適當的取用權限」</b></summary>

通常是自動更新後執行權限沒帶到（v1.4.4 已自動修補，往後不該再發生）。手動修：

1. 開「終端機」
2. 貼上（路徑換成你自己的）：
   ```bash
   chmod +x ~/Desktop/黑貓/黑貓主程式/啟動黑貓工具.command
   ```
3. 重新雙擊即可。打開一次後，新版自動更新會自我修復，以後不會再壞。

</details>

<details>
<summary><b>雙擊 .command 跳出後立刻關掉</b></summary>

通常是 Python 或套件沒裝好。在終端機跑：

```bash
python3 -c "import yaml, pypdf, requests; print('OK')"
```

沒顯示 `OK` 就回到 [步驟 2 / 3](#步驟-2安裝-python-3裝過可跳) 重裝。

</details>

<details>
<summary><b>建單失敗顯示 E009</b></summary>

品名類別已固定為 `0006 3C`。請先確認版本為 v1.5.1 以上，並到「**設定**」分頁按「**儲存設定**」讓設定檔回寫固定值後重試。

</details>

<details>
<summary><b>注音模式 Cmd+V 沒反應</b></summary>

v1.3.4 已修正。如果還是不行，請確認你的版本：程式右上角會顯示 `v1.4.x`。版本太舊就讓它自動更新一次。

</details>

<details>
<summary><b>客戶代號 / API 授權碼怎麼拿</b></summary>

聯絡你的黑貓宅急便業務窗口，要 EGS 企業帳號的 API 資訊。

</details>

<details>
<summary><b>PDF 存到哪裡</b></summary>

`黑貓單號/` 資料夾，跟 `黑貓主程式` 同一層。檔名格式 `訂單號_OBT號碼.pdf`。

</details>

## 隱私與資料

- `config.yaml`（API 帳號）只存在你的電腦，**不會上傳**
- `contacts.json`（通訊錄）只存在你的電腦，**不會上傳**
- 自動更新只下載程式碼，不會碰你的資料檔

此工具僅供 STUDIO A 內部使用。
