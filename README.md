# 黑貓宅急便 企業建單工具

統一速達（黑貓）EGS 企業帳號的桌面建單工具，支援單筆建單、批次 CSV 建單、通訊錄管理，建單後直接輸出 PDF。

---

## 系統需求

- macOS 10.15 以上（Apple Silicon / Intel 均可）
- **Python 3.10 以上**

確認 Python 版本：
```bash
python3 --version
```

---

## 安裝步驟

### 1. 下載程式

點選頁面右上角 **Code → Download ZIP**，解壓縮後放到桌面或你習慣的位置。

### 2. 安裝 Python 套件

開啟「終端機」，進入 `黑貓主程式` 資料夾執行：

```bash
cd ~/Desktop/黑貓/黑貓主程式
pip3 install -r requirements.txt
```

### 3. 設定帳號資料

複製設定範本並填入黑貓給你的帳號資料：

```bash
cp config.yaml.example config.yaml
```

用任何文字編輯器（例如 TextEdit）開啟 `config.yaml`，填入：

| 欄位 | 說明 |
|------|------|
| `username` | 黑貓企業帳號（客戶代號） |
| `api_token` | 黑貓 API 授權碼 |
| `sender.name` | 你的門市／公司名稱 |
| `sender.tel` | 門市電話 |
| `sender.zipcode` | 門市 6 碼郵遞區號 |
| `sender.address` | 門市地址 |

---

## 啟動程式

**方法一：直接雙擊**

雙擊 `黑貓主程式` 資料夾裡的 `啟動黑貓工具.command`

> 首次執行可能出現「無法打開，因為 Apple 無法檢查其是否包含惡意軟體」的提示，請到  
> **系統設定 → 隱私權與安全性** → 點「仍要開啟」即可。

**方法二：終端機**

```bash
cd ~/Desktop/黑貓/黑貓主程式
python3 app.py
```

## 功能說明

| 頁籤 | 說明 |
|------|------|
| 單筆建單 | 填寫一筆收件人資料，立即建立 PDF 託運單 |
| 批次建單 | 上傳 CSV 檔案，一次建立多筆訂單 |
| 通訊錄 | 管理常用收件人，建單時一鍵帶入 |
| 設定 | 修改 API 帳號資料與寄件人資料 |

PDF 檔案會自動存到 `黑貓單號/` 資料夾，檔名格式為 `訂單號_OBT號碼.pdf`。

## 常見問題

**Q：雙擊 .command 沒有反應或立刻關閉？**  
先確認 Python 3 已安裝，然後在終端機執行 `pip3 install -r requirements.txt`。

**Q：建單失敗，出現 E009？**  
請到「設定」頁重新選擇「品名類別」後儲存，再重試。

**Q：API 授權碼在哪裡拿？**  
請聯繫黑貓業務取得 EGS 企業帳號的 API Token。

**Q：PDF 儲存在哪裡？**  
`黑貓單號/` 資料夾（和 `黑貓主程式` 同層）。
